== Installation ==

<strong>Important Note:</strong> Divi Sumo Lite is built using Divi Builder API. This plugin requires Divi Theme, Extra Theme or Divi Builder plugin to work.

1. Uploading the files to your server at wp-content/plugins. Or Upload the plugin zip via your WordPress dashboard.
2. Activate the plugin and done. You should be able to see new divi modules in your Divi editor.
